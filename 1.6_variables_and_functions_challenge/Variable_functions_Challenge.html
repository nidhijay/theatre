Your Challenges:

Challenge 1: Get area from width and length.

  • The code for this challenge has been completed for you as an example. Your challenge is to view the code and try to understand how it works.
  • Lines 3-5 contain the function declaration for the getArea() function, with the parameters of width and length.
  • Line 6 calls that function with the arguments of 4 and 5, and assigns the function's return value to the const variable named area.
  • Finally, line 7 takes the value stored in area, which is 20, and displays it in the first cell of the Challenge Question Results table that you see above the instructions.

Challenge 2: Get perimeter from width and length.

  • First, uncomment lines 16 and 17 - that means to remove the // characters that are at the beginning of each line.
  • Then, complete the unfinished variable declaration on line 16 so that it calls the getPerimeter function (passing in the numbers 4 and 5 as arguments) and saves its return value to the const variable of perimeter.
  • For this and the next challenges, when you are finished, you should see a number appear in the Challenge Question Results table above.
  • If you are not sure how to complete this challenge, look at the code for Challenge #1 to guide you, and ask in the #htmlcssjs channel if you need help.

Challenge 3: Get ounces from pounds.

  • Note: There are 16 ounces in a pound.
  • Write the return statement inside the getOuncesFromPounds function declaration so that it returns the value of pounds * 16.

Challenge 4: Get inches from feet.

  • Note: There are 12 inches in a foot.
  • Declare a function named getInchesFromFeet with a parameter list containing the parameter ft. Inside the function, return the value of ft * 12. Then uncomment the two lines of code below it where indicated.

Challenge 5: Get grams from ounces.

  • Declare a function named getGramsFromOunces with a parameter list containing the parameter ounces. Inside the function, return the value of ounces * 28.3495. Below it, outside the function, call the getGramsFromOunces function with an argument of 5, and save its return value to a const variable named grams. Uncomment the line that begins with document.querySelector... below it.
